## 1. Структура Страниц

### 🏠 Главная (`index.astro`)
*   **Концепция**: "Хаб" реализован. Есть разделение на 3 карточки (Учиться, Кино, Зритель).
*   **Проблема**:
    *   Дизайн базовый (CSS в файле).
    *   Нет "Вау-эффекта", просто навигация.
    *   Карта — заглушка SVG.

### 🎭 Студия (`studio/index.astro`)
*   **Статус**: "Монолит" (885 строк кода!).
*   **Проблема**: Пытается быть всем сразу.
    *   Внутри него живет блок **"Фильмы и награды"** (хотя это контент Киношколы).
    *   Внутри него живет блок **"Кастинг"** (тоже Киношкола).
    *   Смешаны стили: кастомный CSS + попытки использовать компоненты.
*   **Вердикт**: Нужно "распилить". Перенести блоки Фильмов и Кастинга на страницу `/cinema`.

### 🎬 Киношкола (`cinema/index.astro`)
*   **Статус**: Каркас (Skeleton).
*   **Проблема**: Страница практически пустая. Импортируются компоненты (`CastingList`, `SuccessStories`), которых, возможно, еще нет или они пустые.
*   **Стиль**: Использует Tailwind (`bg-black`), в то время как Студия на чистом CSS. Рассинхрон стилей.

## 2. Технический Долг
*   **Дублирование данных**: `films.json` и `vk_films.json` сливаются вручную.
*   **Стили**: Винегрет (Tailwind + Scoped CSS + Global Styles). Нужно выбрать один путь (в Плане мы писали про Tailwind).
*   **Компоненты**: `studio/index.astro` содержит слишком много хардкода (SVG иконок, тексты прямо в верстке), который нужно вынести.
