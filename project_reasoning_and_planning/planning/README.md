## 🚀 Как работать с этими файлами
1.  **Agents.md** — читать перед тем, как делать новый раздел сайта (чтобы попасть в тон).
2.  **Marketing_Strategy.md** — читать перед написанием текстов (чтобы бить в боли клиентов).
3.  **VK_Content_Sync.md** — техническое руководство для программиста (как тянуть данные).
