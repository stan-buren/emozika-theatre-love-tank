# Интеграция с ВКонтакте (VK Content Sync)

## 🎯 Цель
Автоматизировать наполнение сайта контентом. Вместо ручного копипаста, мы "высасываем" данные из живой группы ВК.
Это решит проблему "мертвого сайта", так как ВК обновляется регулярно.

## 🛠 Техническая задача
Написать скрипт/модуль, который через VK API выгружает:
1.  **Фотоальбомы** → в Галерею сайта.
2.  **Видеозаписи** → в раздел Кино (фильмы, трейлеры, бэкстейджи).
3.  **Товары/Услуги** → в раздел Абонементов (цены, описания).
4.  **Посты (Стена)** → в Новости/Блог или "Награды" (если фильтровать по тегам).

## 💾 Архитектура (Предложение)
Так как у нас Astro (статический сайт), лучшая схема — **Build-Time Fetching**.

1.  **Скрипт-сборщик:** `src/scripts/fetch-vk-data.js`
    *   Запускается перед билдом (или вручную).
    *   Ходит в VK API с сервисным токеном.
    *   Скачивает JSON-данные и сохраняет их в `src/data/`.
    *   Скачивает картинки (опционально, или использует прямые ссылки VK, но лучше кэшировать).
    
2.  **Сайт:**
    *   При сборке читает `src/data/*.json`.
    *   Генерирует красивые страницы.

**Плюсы:**
*   Сайт летает (это чистый HTML).
*   Не нужна сложная база данных (MySQL/Postgres).
*   Бесплатно хостится.
*   Обновление контента = перезапуск билда (можно настроить по кнопке или расписанию).

## 📋 План действий
1.  [ ] **Доступ:** Получить Service Token (создать Standalone App в VK Developers).
2.  [ ] **Анализ данных:** Посмотреть JSON-ответы от ВК. Понять, как отличать "Награду" от "Просто поста". (Хэштеги? Отдельный альбом?).
3.  [ ] **Прототип:** Скрипт, который тянет 10 последних постов/видео.
4.  [ ] **Интеграция:** Подключить json к компонентам Astro.
