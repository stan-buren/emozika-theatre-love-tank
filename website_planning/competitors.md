# 🕵️ Анализ Конкурентов (Competitor Analysis)

## 1. Talantino (Прямой конкурент)
**Ссылка:** [talantino.ru](https://talantino.ru)
**Статус:** "Эталон воронки продаж"
*   **Сильные стороны:**
    *   **Фокус на Агентстве:** Четко отделено Агентство (для профи) от Студии (для обучения). 
    *   **Тон:** Деловой, серьезный. "Мы работаем с кастинг-директорами".
*   **Что берем:**
    *   Логику разделения Студия vs Агентство.
    *   Агрессивный CTA для кастинга.

## 2. Moscow Film School (Референс "Мечта")
**Ссылка:** [moscowfilmschool.ru](https://moscowfilmschool.ru)
**Статус:** "Тяжелый люкс в образовании"
*   **Сильные стороны:**
    *   **Типографика и Воздух:** Огромные заголовки, миниализм. Ощущение дорогого образования.
    *   **"Истории успеха":** Выпускники, которые достигли высот (лучшее доказательство).
*   **Что берем:**
    *   Стиль заголовков (крупный sans-serif).
    *   Раздел "Истории успеха" для нашего Киноагентства.

## 3. Kino.School (Визуальный Референс)
**Ссылка:** [kino.school](https://kino.school/)
**Статус:** "Энергия и Молодость"
*   **Сильные стороны:**
    *   **Цвет:** Глубокие, "киношные" градиенты (фиолетовый, неон).
    *   **Динамика:** Сайт живой, много микро-анимаций.
*   **Что берем:**
    *   Цветовую палитру (наш фиолетовый нужно увести в "глубину").
    *   Стиль кнопок и карточек.

## 4. Deti.Media (Анти-пример)
**Ссылка:** [deti.media/castings](https://deti.media/castings)
**Статус:** "Доска объявлений 2010"
*   **Проблемы:** Визуальный шум, нет иерархии, дизайн а-ля Windows 98.
*   **Вывод:** Наша страница `/cinema` должна быть *кардинально* чище.

## 5. McGuffin (Смыслы)
**Ссылка:** [mcguffin.ru](https://mcguffin.ru)
**Статус:** "Хороший продукт, слабая упаковка"
*   **Что берем:** Названия форматов. "Киношкола" звучит лучше, чем "Кружок".
